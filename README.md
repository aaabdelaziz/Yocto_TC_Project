# Yocto_TC_Project – Temperature-Controlled Fan on Raspberry Pi 4

This repository contains the Yocto configuration and application code for my final project in the **Linux Embedded System Topics and Projects** course. The project implements a temperature-controlled fan system running on a Raspberry Pi 4, with an optional Flask-based web interface to monitor temperature and fan status.

For a complete description of the project goals, hardware, build system, and schedule, see the Project Overview wiki page:
